Download

First you need to have the library core files (joint.js and joint.css - or their minified versions). Plugins can be intermixed arbitrarily or not used at all. Note that the library core file (joint.js or joint.min.js) is already packed with jQuery, Backbone and Underscore libraries. If you already have one of those libraries included in your application, you might want to use a JointJS build without such libraries included. Please go to the dist directory on Github to download a build of JointJS without the library that you don't need.

Development version (0.9.2)

joint.js joint.css

Production version (0.9.2)

joint.min.js joint.min.css

Clean version (less of jQuery, Backbone, Lodash, Geometry and Vectorizer) (0.9.2)

This is useful in cases where you want to use your own jQuery, Backbone or Lodash libraries, or if you plan to integrate JointJS with RequireJS.

joint.clean.js joint.clean.min.js

Note that this library is already packaged with the joint.js file. This download is only meant for a standalone use of the Geometry library.
