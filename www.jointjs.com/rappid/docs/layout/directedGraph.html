layout.DirectedGraph

DirectedGraph auto-lays out directed graphs.

Installation

Include joint.layout.DirectedGraph.js file to your HTML:


<script src="joint.layout.DirectedGraph.js"></script>

Usage

DirectedGraph plugin exposes one function joint.layout.DirectedGraph.layout(graph, options). The first parameter graph is the graph we want to layout. The second parameter options is an object that contains various options for configuring the layout.

Options

  • debugLevel - verbosity level of the layout engine - useful for debugging
  • nodeSep - a number of pixels representing the separation between adjacent nodes in the same rank
  • edgeSep - a number of pixels representing the separation between adjacent edges in the same rank
  • rankSep - a number of pixels representing the separation between ranks
  • rankDir - direction of the layout (one of "TB" (top-to-bottom) / "BT" (bottom-to-top) / "LR" (left-to-right) / "RL" (right-to-left))

Additionally, the layout engine takes into account minLen attribute of links that represents the number of ranks to keep between the source and target of the link.

The layout() function returns an object that contains width and height of the resulting layouted graph.


var res = joint.layout.DirectedGraph.layout(graph, {
    nodeSep: 50,
    edgeSep: 80,
    rankDir: "TB"
});
console.log(res.width, res.height);
