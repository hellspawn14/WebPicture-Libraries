Rappid A complete diagramming toolkit.

Use Rappid to create advanced visual tools in days, not months! Focus on your underlying business logic instead of the UI.

Rappid is a set of JointJS plugins and other components that, when combined, extremely speed up development of advanced applications. Rappid takes the best of HTML 5 and SVG to give you the right tools for building your cloud products.

What you get with Rappid

  • A commercial license to both Rappid and JointJS core library (including source code).
  • A set of more than 20 plugins giving you a complete toolkit for building diagramming applications.
  • Three applications (one simple to get you started quickly, another one, more complete, that you can see here) and a bonus BPMN (Business Process Model and Notation) modeling application.

Rappid plugin set

Halo

A configurable control panel above elements with important tools right at hand.

Stencil

Element pallete with animated accordion-like grouping.

FreeTransform

Resize your element to all the sides, even when rotated.

Channel

Bring real-time collaboration to your applications.

Clipboard

Copy, cut and paste with HTML 5 localStorage support for copy-pasting across page refreshes.

Selection

Manipulate more elements in one go. Supports both bulk and individual element selection.

Charts

Plot both Line and Bar charts with ease. Charts are just another JointJS elements so that you can use them as any other shape.

Validator

Make sure your documents are always in a state you want them to be. Automatically cancel invalid actions.

PaperScroller

A smaller view to a larger paper area with scroll & pan, centering, auto-resizing/shrinking and zooming.

Inspector

An extremelly flexible and completely configurable properties editor and viewer.

Undo/Redo

Undo/redo to an arbitrary level.

Tooltip

Display your info messages anywhere in the UI. Supports HTML content too.

GridLayout

Layout your elements automatically in a grid. Configurable columns, rows, their dimensions and paddings.

DirectedGraph layout

Auto-layout your directed graphs. Configurable.

ForceDirected layout

Force-directed auto-layout. Uses ressemblance to the physical world to make sure the resulting graph is appealing.

Print

Print your documents via the browser print dialog box.

SVG Export

Export your documents into the SVG vector format with one function call.

Raster Export

Export your documents into the PNG or JPEG format.

GEXF Import

Import graphs from the GEXF XML format.

Shapes

A set of ready-to-use shapes of well know diagrams: FSA, ERD, UML, PN, ORG, DEVS.

License

The Rappid license works on a per-developer basis. Each engineer developing against the Rappid plugins API needs a single license. The license then gives you the freedom to use Rappid to create an unlimited number of commercial applications, use the plugins on any computer or server you like and you'll have access to the source code of Rappid and its plugins with the freedom to modify it.
